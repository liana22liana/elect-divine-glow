

## Доработка страницы /profile и системы амбассадоров

### Обзор
Полная переработка профиля: аватар, блок подписки, визуальный timeline статусов амбассадора с подарками, кнопка поддержки. Плюс обновление админки (управление статусами) и библиотеки (закрытый контент). На моковых данных.

### Изменения

**1. Данные (`src/lib/mock-data.ts`)**
- Расширить `UserProfile`: `subscription_status` (active/paused/cancelled), `subscription_start_date`, `subscription_end_date`, `ambassador_status` (null/rising/becoming/transformed/reborn), `ambassador_status_override`, `delivery_form_submitted`, `avatar_url`
- Новые интерфейсы: `AmbassadorGift`, `LockedContent`
- Моковые данные: подарки по этапам (2/3/6/12 мес.), закрытые материалы
- Константа `AMBASSADOR_MILESTONES` — массив этапов с названиями, месяцами, описаниями подарков, типами

**2. Профиль (`src/pages/ProfilePage.tsx`) — полная переработка**
- **Шапка**: аватар (круг с кнопкой загрузки), имя, "В клубе с [дата]"
- **Блок подписки**: статус с цветовой индикацией (зелёный/жёлтый/красный), дата списания, кнопка "Управление подпиской"
- **Timeline амбассадора**: горизонтальный на десктопе, вертикальный на мобильном
  - 4 этапа: Восходящая (2м) → Становящаяся (3м) → Трансформированная (6м) → Возрождённая (12м)
  - Достигнутые = золотой + галочка, текущий = пульсирующая анимация, будущие = бледные + замок
  - Под каждым: описание подарка
  - Прогресс-бар между этапами: "X из Y дней"
- **Модалка поздравления**: при клике на достигнутый этап — "Поздравляем! Ты — [статус]" + описание подарка + кнопка действия:
  - 2 мес.: кнопка "Вступить в ТГ-канал" (ссылка)
  - 3 мес.: "Открыть закрытый материал" (ссылка на материал)
  - 6/12 мес.: форма доставки (имя, телефон, email, адрес) → "Даша свяжется с тобой в течение 3 дней 🤍", скрывается после отправки
- **Кнопка поддержки**: фиксированная внизу, иконка Telegram, ссылка на https://t.me/daavilovahelp
- Кнопка "Выйти из аккаунта" остаётся

**3. Компонент Timeline (`src/components/AmbassadorTimeline.tsx`)**
- Отдельный компонент для переиспользования
- Адаптивный: горизонтальный (md+) / вертикальный (mobile)
- Линии-соединители между этапами с прогресс-заливкой

**4. Модалка поздравления (`src/components/AmbassadorCongratDialog.tsx`)**
- Анимация свечения при открытии
- Форма доставки для физических подарков (6/12 мес.)

**5. Библиотека — закрытый контент**
- `MaterialCard.tsx`: если материал в `lockedContent` и статус пользователя недостаточен — замок поверх карточки + tooltip "Открывается при статусе [название]"
- `LibraryPage.tsx`: фильтрация/отображение с учётом locked-статуса

**6. Админка (`src/pages/AdminPage.tsx`) — вкладка "Участницы"**
- Раскрываемая карточка участницы с:
  - Текущий статус амбассадора (авто/ручной)
  - Switch "Присвоить вручную" → Select статуса, пометка "Присвоено вручную"
  - Поле ссылки-приглашения в ТГ-канал
  - Select закрытого материала для 3 мес.
  - Статус физического подарка (не заполнена / заполнена / отправлен)
