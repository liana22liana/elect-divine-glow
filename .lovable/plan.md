

## Обновление структуры библиотеки контента

### Обзор
Переход от плоских категорий к разделам с подразделами, добавление дополнительных материалов к карточкам, и управление структурой в админке. Всё на моковых данных (подготовлено к Supabase).

### Изменения

**1. Данные (`src/lib/mock-data.ts`)**
- Новые интерфейсы: `LibrarySection`, `LibrarySubsection`, `AdditionalMaterial`
- Обновить `Material`: заменить `category: string` на `section_id` и `subsection_id`
- Добавить `additional_materials` к каждому материалу
- Моковые разделы (7 шт.) с подразделами для "Тело" (Эфиры, Кундалини, Тренировки, Питание) и "Практики и медитации" (Медитации, Практики)
- Удалить старый `CATEGORIES` — заменить на `LIBRARY_SECTIONS`

**2. Библиотека (`src/pages/LibraryPage.tsx`)**
- Верхняя навигация: горизонтальные вкладки по разделам
- При выборе раздела с подразделами — под основными вкладками появляется вторая строка вкладок-подразделов
- При выборе раздела без подразделов — сразу сетка карточек
- URL-параметры: `?section=body&sub=workouts`

**3. Карточка материала (`src/components/MaterialCard.tsx`)**
- Обновить отображение тега: раздел вместо категории

**4. Страница материала (`src/pages/MaterialPage.tsx`)**
- Добавить секцию "Дополнительные материалы" после описания
- Список с иконкой типа (Video/Headphones), названием и кнопкой воспроизведения
- При клике — открытие в модалке или переход по ссылке

**5. Админка (`src/pages/AdminPage.tsx`)**
- Новая вкладка "Структура" (между "Материалы" и "Участницы")
  - Список разделов с кнопками: переименовать, удалить, drag-handle для порядка
  - Раскрытие раздела показывает подразделы (аналогично)
  - Кнопки "Добавить раздел" / "Добавить подраздел"
- Обновить диалог создания материала: Select для раздела → Select для подраздела (зависимый), секция добавления доп. материалов (название, тип, ссылка)

**6. Обновить зависимые компоненты**
- `CategoryCard.tsx` → обновить на новую структуру разделов
- `HomePage.tsx` → использовать `LIBRARY_SECTIONS` вместо `CATEGORIES`
- `HabitCard.tsx`, `AddHabitDialog.tsx` → обновить ссылки на категории
- `GoalsPage.tsx` — если использует `CATEGORIES`
