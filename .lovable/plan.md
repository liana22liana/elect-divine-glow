

## Трекер привычек для ELECT

### Что будет сделано

Полноценный модуль "Трекер привычек" с пользовательской страницей `/goals` и админским разделом в `/admin`. Пока на моковых данных (без Supabase), в стиле платформы.

### Структура изменений

**1. Данные и типы** (`src/lib/mock-data.ts`)
- Добавить интерфейсы: `HabitTemplate`, `Habit`, `HabitLog`
- Моковые шаблоны от админа (3-4 рекомендации, привязанные к существующим материалам)
- Моковые привычки пользователя (3 штуки: ежедневная, еженедельная, с дедлайном)
- Моковые логи выполнения за последний месяц

**2. Страница участницы** (`src/pages/GoalsPage.tsx` — маршрут `/goals`)
- Шапка: прогресс за месяц (Progress bar "Выполнено X% от плана") + мотивационная фраза
- Карточки привычек:
  - Название + иконка категории
  - Ссылка на материал (если из библиотеки)
  - Частота ("каждый день" / "3 раза в неделю")
  - Для привычек с дедлайном: прогресс-бар "X из Y"
  - Для бессрочных: мини-календарь последних 4 недель (grid 7x4, закрашенные ячейки)
  - Стрик "5 дней подряд"
  - Кнопка "Отметить сегодня" с pulse-анимацией при нажатии
- FAB-кнопка "+" для добавления привычки

**3. Модальное окно добавления** (`src/components/AddHabitDialog.tsx`)
- Две вкладки (Tabs): "Своя привычка" / "Из клуба"
- Вкладка "Своя": поле названия, переключатель частоты (ежедневно / еженедельно / N раз в неделю со слайдером), переключатель дедлайна (бессрочная / до даты с DatePicker + поле "всего раз")
- Вкладка "Из клуба": список рекомендаций-карточек с кнопкой "Добавить в мои цели" → открывает настройку частоты/дедлайна

**4. Компонент карточки привычки** (`src/components/HabitCard.tsx`)
- Отдельный компонент для переиспользования
- Мини-календарь как grid 7 колонок x 4 строки с цветовым кодированием

**5. Админ-панель** — новая вкладка в `AdminPage.tsx`
- Третья вкладка "Рекомендации" рядом с "Материалы" и "Участницы"
- Таблица рекомендаций: название, категория, привязанный материал, дата
- Кнопка "Добавить рекомендацию" → диалог с полями: название, описание, категория (Select), привязка к материалу (Select с поиском по mockMaterials)
- Кнопки редактирования/удаления с подтверждением
- Мини-статистика: "добавили X участниц"

**6. Навигация**
- `BottomNav.tsx`: добавить пункт "Цели" (иконка `Target`) между "Библиотека" и "Профиль"
- `DesktopSidebar.tsx`: добавить пункт "Цели" с иконкой `Target`
- `App.tsx`: добавить маршрут `/goals` → `GoalsPage`

### Дизайн
- Бежево-золотая палитра платформы, акценты `#8A9EA6` для второстепенных элементов (мини-календарь незаполненных дней, подписи)
- Карточки: `rounded-2xl`, легкая тень, `border-border`
- Кнопка "Отметить" — золотая (`bg-primary`), при нажатии `animate-scale-in`
- Стрик с эмодзи огня
- Мини-календарь: маленькие круги, заполненные = `bg-primary`, пустые = `bg-muted`
